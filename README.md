# RAJINTALKSALOT

Fresh project workspace for RAJINTALKSALOT.

## Notes

- Project type: Astro static web app
- Package manager for any web app work: pnpm

## Web App

```bash
cd web
pnpm install
pnpm run optimize:images
pnpm run dev
pnpm run build
```

The app stores source images in `rajintalksalot/posts` and optimized WebP/AVIF files in `web/public/media`.
